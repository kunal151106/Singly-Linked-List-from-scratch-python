# Python Linked List

A simple implementation of a **Singly Linked List** in Python built using object-oriented programming. This project demonstrates the core concepts of linked lists by implementing common operations from scratch.

## Features

* Append new nodes to the end of the list
* Prepend nodes to the beginning
* Insert nodes at a specific index
* Delete nodes by index
* Remove nodes by value
* Replace values at a given index
* Search for elements in the list
* Find the index of a value
* Count occurrences of a value
* Check if the list is empty
* Get the current length of the list
* Find the maximum and minimum values
* Reverse the linked list in place
* Clear all nodes from the list
* Display the linked list in a readable format

## Purpose

* Learn the fundamentals of linked lists
* Practice Data Structures and Algorithms (DSA)
* Understand pointer-based data structures
* Strengthen Python and Object-Oriented Programming (OOP) concepts
* Prepare for coding interviews

## Technologies Used

* Python 3
* Object-Oriented Programming (OOP)
